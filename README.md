# Laravel E-commerce Core API 

## Overview
A scalable Backend API for E-commerce platforms, designed with a focus on **High Availability** and **Clean Architecture**. This project showcases advanced Laravel features and architectural patterns.

## Technical Highlights
* **Service Layer Pattern**: Managed complex business logic for checkout and inventory.
* **Factory Design Pattern**: Used for dynamic discount and tax calculations.
* **Database Optimization**: Advanced indexing for product searching and filtering.
* **Security**: Role-based access for Admins and Customers using Laravel Sanctum.

## Core Modules
* Product & Category Management.
* Shopping Cart Logic (Database/Redis-backed).
* Order Processing & Status Tracking.
